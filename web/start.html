Bewertung von Szenarien

Wählen Sie nun bitte zwischen gleichfarbigen Filmgruppen aus und bestätigen Sie Ihre Auswahl mit einem Klick auf das vorgesehene Feld. Halten Sie beide Filmgruppen für ansprechend, können Sie beide Felder anklicken.

Aller Filmgruppen auf der linken Seite sind der ersten Methode zuzuordnen, Filmgruppen auf der rechten Seite gehören der zweiten Methode an.

Unterhalb der Filmgruppen finden geben Sie Ihre Bewertung zu der jeweiligen Methode ab. Klicken Sie auf den linken Button, so bevorzügen Sie die erste Methode. Durch einen Klick auf den rechten Button ziehen Sie die zweite Methode vor. Sollten Sie sich zwischen beiden Methoden nicht entscheiden können, so wählen Sie den mittleren Button.
